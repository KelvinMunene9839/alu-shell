0-current_working_directory: Prints the absolute path of the current working directory.
1-listit: Displays the contents of the current directory.
2-bring_me_home: Changes the current directory to the user's home directory.
3-listfiles: Displays the current directory contents in long format.
4-listmorefiles: Displays the current directory contents in long format, including hidden files.
5-listfilesdigitonly: Displays the current directory contents in long format with numeric user and group IDs, including hidden files.
6-firstdirectory: Creates /tmp/my_first_directory.
7-movethatfile: Moves /tmp/betty into /tmp/my_first_directory.
8-firstdelete: Deletes /tmp/my_first_directory/betty.
9-firstdirdeletion: Deletes /tmp/my_first_directory.
10-back: Changes to the previous working directory.
11-lists: Lists the current directory, its parent, and /boot in long format, including hidden files.
12-file_type: Prints the file type of /tmp/iamafile.
13-symbolic_link: Creates a symbolic link named __ls__ that points to /bin/ls.
14-copy_html: Copies HTML files to the parent directory when they are new or newer.
15-lets_move: Moves files beginning with an uppercase letter to /tmp/u.
16-clean_emacs: Deletes files ending with ~ in the current directory.
17-tree: Creates welcome/to/school with parent directories as needed.
